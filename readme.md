<h2 id="sobre"> :pencil: Sobre</h2>

<p align="justify"> 
Neste trabalho nos foi dado o objetivo de simular diferentes partidas de tênis representando o jogo por meio de uma cadeia de Markov, para então observar seu comportamento baseado na diferença da probabilidade de vitória de seus oponentes.
</p>
